# DCM

## Hướng Dẫn

Khi công việc liên quan đến HTML template bảo hiểm DCM trong `src/assets/html_templates`, đọc và làm theo `references/html-template.md` trước khi phân tích hoặc chỉnh sửa.

Các tác vụ liên quan bao gồm:

- Chỉnh sửa, tạo mới, review hoặc debug template `GCN_*` / `GYC_*`.
- Thay đổi `index.html`, `style.css`, `sample_data.json`, `fields.md` hoặc file Tailwind của template.
- Xử lý field Jinja, layout PDF/in ấn, CSS trang A4 hoặc asset riêng của template.

Giữ thay đổi theo đúng phạm vi và quy ước trong file tham chiếu.
